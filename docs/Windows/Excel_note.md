## VLOOKUP
Excel テーブルちっくなの
```
VLOOKUP(検索値, 範囲, 列番号 [, 検索方法])


（例）
=VLOOKUP(E5, R:S, 2, FALSE)


第４引数 : TRUE  - 近似一致（※範囲の先頭列を昇順で並べ替えておく必要がある）
           FALSE - 完全一致

デフォルトの TRUE は、かなりいい加減で使い物にならないので、FALSE を指定しておいた方がいい。
```
